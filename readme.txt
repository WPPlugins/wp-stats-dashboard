== Description ==
Displays the WordPress.com stats graph directly in your Dashboard.
No more tiresomely opening the Blog Stats tab and manually logging in.

Notable features:

1. Display the WordPress.com stats graph in your dashboard.
2. Choose the type of stats graph you want to display.
3. Automatic Login.
4. Stats Dashboard Widgets. (graph, overview, clicks, referrers, post views and search terms).

== Installation ==

1. Extract zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Requires CURL extension to be active in php configuration. (Just try the plugin and you'll know)
4. Requires wp-content/cache to be writable by webserver.
